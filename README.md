# Data-Analytics-Dashboard
📊 Sales Data Analytics Project (Python + Pandas)
1️⃣ Project Overview

This project is a Sales Data Analytics System developed using Python and Pandas.
The main objective of the project is to analyze sales data from a CSV file and generate meaningful business insights such as:

Total Revenue

Monthly Sales Performance

Top Selling Products

The system processes raw sales data and converts it into structured business information that helps management make data-driven decisions.

2️⃣ Technologies Used

Python

Pandas Library

CSV File Handling

DateTime Processing

Data Grouping & Aggregation

3️⃣ Project Architecture / Workflow

Step-by-Step Flow:

1.CSV file is uploaded

2.Data is read using Pandas

3.Data cleaning & transformation is performed

4.Business calculations are done

5.Results are returned in dictionary format
